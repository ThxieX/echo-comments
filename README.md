
# 📡 echo-comments

> **"What is a soul? It's like an echo."**

This repository serves as the decentralized comment engine for my digital garden.  
Powered by **Giscus** and **GitHub Discussions**.

---

## 🛠 Project Purpose
- **Decoupling**: Separating user-generated content from source code.
- **Persistence**: Leveraging GitHub's infrastructure for long-term data reliability.
- **Experience**: Markdown-native, developer-friendly, and zero-tracker environment.

## ⚙️ Configuration
- **Target Site**: [thxie.com](https://thxie.com)
- **Engine**: [Giscus](https://giscus.app)
- **Mapping**: `pathname`
- **Theme**: Gruvbox-first multi-theme

## 🛡️ Governance
All comments are stored as **Discussions**. I reserve the right to prune spam or low-density content to maintain the signal-to-noise ratio.
